# The AI Mind

**From First Principles to AI Researcher**

《The AI Mind》是一套以中文为主、面向长期维护的开放式 AI 教材工程。目标不是堆砌术语，而是通过直觉、数学、代码、工程与研究，培养达到美国名校 AI 硕士毕业生水准的系统能力。

## 四本书

1. **Book I · Discovering Intelligence**：AI 思维、表示、计算、学习与泛化
2. **Book II · Learning to Learn**：线性代数、概率、优化与经典机器学习
3. **Book III · Learning to Build**：micrograd、makemore、Transformer、GPT 与 LLM 工程
4. **Book IV · Learning to Create**：论文阅读、AI Systems、Agent 与毕业项目

## 教学链条

每个重要知识点遵循：

> 故事 → 费曼解释 → 直觉 → 第一性原理 → 数学 → 代码 → 工程 → 金融应用 → 研究问题
