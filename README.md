# The AI Mind

**From First Principles to AI Researcher**

《The AI Mind》不是一本书，而是一套持续演进、开放协作、基于第一性原理的 AI 课程体系。

The AI Mind is a living, open curriculum for learning artificial intelligence from first principles. It is designed to help curious learners grow into AI engineers, researchers, and builders.

New contributor or AI assistant? Start with [MANIFEST.md](MANIFEST.md).

## 四本书

1. **Book I · Discovering Intelligence**：AI 思维、表示、计算、学习与泛化
2. **Book II · Learning to Learn**：线性代数、概率、优化与经典机器学习
3. **Book III · Learning to Build**：micrograd、makemore、Transformer、GPT 与 LLM 工程
4. **Book IV · Learning to Create**：论文阅读、AI Systems、Agent 与毕业项目

## 教学链条

每个重要知识点遵循：

> 故事 → 费曼解释 → 直觉 → 第一性原理 → 数学 → 代码 → 工程 → 金融应用 → 研究问题
