# Editorial Reviews

This directory stores permanent editorial review reports for The AI Mind.

Reviews are not side notes. They are part of the publication history.

## Roles

| Role | Responsibility |
|---|---|
| Codex / Author | Draft chapters, notebooks, figures, glossary updates, commits |
| ChatGPT / Editor-in-Chief | Editorial review, technical correctness, pedagogy, narrative, research depth |
| GitHub | Single source of truth |

## Standard Flow

```text
Draft (Author)
  ↓
Editorial Review (Editor)
  ↓
Revision (Author)
  ↓
Approval (Editor)
  ↓
Merge / publish
```

## Directory Layout

```text
reviews/
  REVIEW_TEMPLATE.md
  book1/
    chapter16_review.md
    chapter17_review.md
```

Each review should be committed and preserved.

## Naming

Use lowercase filenames:

```text
chapter16_review.md
chapter17_review.md
```

Revision notes may use:

```text
chapter16_revision_notes.md
```
