# Contributing

感谢参与《The AI Mind》。

## 可贡献内容

- 修正事实、公式或代码错误
- 改进费曼解释和直觉
- 增加图示、练习和 Notebook
- 补充引用和反例
- 改进中文表达

## Pull Request 要求

- 每个 PR 聚焦一个明确问题
- 说明修改原因及读者收益
- 数学或事实修改需给出可靠来源
- 代码需可运行，并说明验证方式
- 避免在同一 PR 中进行无关的大规模格式调整

## Issue 标签建议

- `content`
- `math`
- `code`
- `editorial`
- `figure`
- `exercise`
- `research`
