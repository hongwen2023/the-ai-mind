# Project Operations

Mission: Maintain The AI Mind as a long-term open curriculum.

## Operational Principles

- Repository is the single source of truth.
- One completed chapter per merge.
- Continuous documentation.
- Continuous review.
- Automate wherever possible.

## Publishing Workflow

The core production model is:

```text
Author draft
  ↓
Editor review
  ↓
Author revision
  ↓
Approval
  ↓
Publish
```

Reviews are stored in `reviews/` and should be treated as part of the project history.
