# The AI Mind --- Author Bible

## Purpose

This document defines the editorial philosophy for every chapter.

## Audience

Primary reader: a motivated learner targeting the capability of a top US
AI Master's graduate.

## Writing Philosophy

Story → Feynman → Intuition → First Principles → Mathematics → Code →
Engineering → AI × Finance → Research.

Golden Rule: **Build intuition before precision.**

## Never Do

-   Never start with equations.
-   Never introduce jargon without motivation.
-   Never separate theory from engineering.
-   Never end without connecting to the next chapter.
