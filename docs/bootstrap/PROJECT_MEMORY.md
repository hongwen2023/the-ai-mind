# PROJECT_MEMORY

## Long-term Vision

The AI Mind is not merely a book. It is a living AI curriculum that
evolves over many years.

## Target Reader

A learner who wants to reach the capability of a top US AI Master's
graduate.

## Editorial Style

Story → Feynman → Intuition → First Principles → Mathematics → Code →
Engineering → AI × Finance → Research.

## Non-negotiables

-   Never start with equations.
-   Always build intuition before precision.
-   Every chapter ends with Research Corner.
-   One chapter per Markdown file.
