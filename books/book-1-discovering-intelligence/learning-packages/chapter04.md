## Mental Model Upgrade

```text
Before: representation = reality converted into numbers.

After:  representation = observation boundary
        + encoding choice
        + geometry
        + available operations
        + declared information loss.
```

The upgrade is complete only when the learner can explain which distinctions
the model can no longer recover, how encoding changes the task geometry, and
why a task-suitable representation may intentionally discard reality details.

## Capability Milestone

After this package, the learner can:

- [ ] **Explain** Reality, Observation, Representation, and Model.
- [ ] **Predict** geometry changes caused by a different encoding.
- [ ] **Build** and compare multiple encodings for one semantic object.
- [ ] **Read** an AI or finance pipeline for collision, leakage, and false geometry.
