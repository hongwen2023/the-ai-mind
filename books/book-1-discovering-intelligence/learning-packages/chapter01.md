# Chapter 1 Learning Package

## Chapter

[为什么理解比记忆重要？](../chapters/chapter-01.md)

## Learning Outcomes

This package trains one reusable assessment method:

```text
Explain → Predict → Reconstruct → Transfer
```

The chapter introduces the relationship; the Notebook makes rule generation
observable; the Audit collects evidence of understanding; the figures provide
retrieval anchors.

## Notebook

[Lookup Table vs Rule Model](../../../notebooks/book1/chapter01_lookup_vs_rule.ipynb)

Use it to compare replaying stored answers with generating a falsifiable
prediction from a relationship.

## Suggested Order

1. Read the chapter through Mathematics.
2. Predict the Notebook results before execution.
3. Finish the chapter and Common Illusions.
4. Complete the Understanding Audit.
5. Return after one week for reconstruction.

## Completion Evidence

- [ ] I can explain why memory is necessary but insufficient.
- [ ] I can predict how a perturbation changes a system.
- [ ] I can reconstruct the four tests from a blank page.
- [ ] I can transfer the tests to a finance or non-AI problem.
- [ ] I can state where my current relationship model may fail.
