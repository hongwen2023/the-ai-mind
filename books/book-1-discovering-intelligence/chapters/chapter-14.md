## Opening Story: 现实不是直线

假设你在分析一家公司的收入增长。

一开始，收入每年稳定增长：

```text
Year 1: 100
Year 2: 120
Year 3: 140
```

你可能会觉得，这是线性的。

但突然，公司产品突破了临界规模，网络效应开始出现：

```text
Year 4: 190
Year 5: 310
Year 6: 520
```

世界突然不再像一条直线。

再过几年，市场接近饱和，增长又开始放缓：

```text
Year 7: 650
Year 8: 720
Year 9: 760
```

这条曲线有三个阶段：

- 起步；
- 加速；
- 饱和。

这不是一条直线。

现实世界中，很多最重要的现象都是非线性的：

- 学习曲线；
- 技术扩散；
- 用户增长；
- 风险爆发；
- 神经元激活；
- 市场崩盘；
- 语言理解。

如果 AI 只能做线性变换，它就只能理解一个过度简化的世界。

## The One Sentence

> 线性负责重组表示，非线性负责弯曲空间；没有非线性，深度网络再深也只是一个线性模型。

## Why Before What

上一章我们看到，多个线性变换可以合并：

```text
y = W3 @ W2 @ W1 @ x
```

等价于：

```text
y = W @ x
```

这意味着，如果神经网络只有线性层，深度不会带来本质表达力。

那么问题来了：

> 怎样让每一层不只是重新混合特征，而是真的创造新的形状？

答案是加入非线性。

神经网络的基本节奏因此变成：

```text
Linear
  ↓
Activation
  ↓
Linear
  ↓
Activation
```

其中 Activation 就是非线性进入模型的入口。

## Feynman Explanation

想象你在做菜。

线性组合像是按比例混合原料：

```text
0.5 * 盐 + 0.2 * 糖 + 0.3 * 酱油
```

这当然有用。

但真正改变味道的，往往不是简单混合，而是加热、发酵、焦化、凝固。

这些过程不是线性的。

鸡蛋加热到某个温度才凝固。

糖加热到某个程度才焦化。

面团发酵超过某个时间才膨胀。

非线性就像这些“变化规则”。它让系统出现阈值、转折、饱和和选择。

AI 也是一样。

线性层负责混合信号。

非线性负责决定：

> 哪些信号应该通过，哪些应该被压低，哪些应该被截断，哪些应该被强调。

## Historical Background

早期神经网络的灵感来自生物神经元。

一个神经元接收许多输入信号。只有当信号强度超过某个阈值时，它才会明显激活。

这本身就是非线性。

如果神经元只是线性累加输入，那么无论堆多少层，网络都无法表达复杂决策边界。

后来，神经网络使用过多种激活函数：

- Step function；
- Sigmoid；
- Tanh；
- ReLU；
- GELU；
- SwiGLU。

每一次激活函数的变化，都对应着训练稳定性、表达能力和工程效率的权衡。

ReLU 的流行不是偶然。它简单、计算快、梯度传播相对稳定，帮助深度神经网络在工程上变得可训练。

现代 Transformer 常用 GELU、SwiGLU 等更平滑或更强表达力的激活结构。

## First Principles

非线性的核心作用是打破线性闭包。

如果没有非线性：

```text
h1 = W1 @ x
h2 = W2 @ h1
y  = W3 @ h2
```

最终仍然是：

```text
y = W @ x
```

加入非线性后：

```text
h1 = activation(W1 @ x)
h2 = activation(W2 @ h1)
y  = W3 @ h2
```

此时不能再合并成一个矩阵。

每一层都先改变坐标，再弯曲空间，再改变坐标，再弯曲空间。

这让模型可以逐步构造复杂函数。

## Intuition: 折叠空间

线性变换像拉伸橡皮纸。

非线性像折叠橡皮纸。

如果两个点原本很远，通过折叠，它们可以变近。

如果两个点原本很近，通过折叠，它们也可以被分开。

分类问题常常可以这样理解：

> 模型通过一层层变换，把原本纠缠在一起的数据折叠成容易分开的形状。

线性层改变视角。

非线性改变形状。

## Mathematics: ReLU 的简单力量

最常见的激活函数之一是 ReLU：

```text
ReLU(x) = max(0, x)
```

它的规则非常简单：

```text
如果 x > 0，输出 x
如果 x <= 0，输出 0
```

它看起来粗糙，但很有力量。

ReLU 做了一个选择：

> 只有正信号通过，负信号被截断。

这让网络可以形成分段线性函数。

许多 ReLU 单元组合起来，可以拼出非常复杂的曲线和边界。

这就是一个重要直觉：

> 深度网络不是用一个复杂函数解决问题，而是用许多简单的分段函数拼出复杂函数。

## Common Activation Functions

| Activation | 直觉 | 优点 | 局限 |
|---|---|---|---|
| Sigmoid | 把数压到 0 到 1 | 像概率 | 容易梯度消失 |
| Tanh | 把数压到 -1 到 1 | 输出居中 | 仍可能饱和 |
| ReLU | 正数通过，负数截断 | 简单高效 | 可能出现 dead ReLU |
| GELU | 平滑地选择信号 | Transformer 常用 | 计算更复杂 |
| SwiGLU | 门控式特征选择 | LLM 常用 | 参数和结构更复杂 |

不要死记这些函数。

更重要的是理解：

> 激活函数决定模型如何筛选、压缩、放大和弯曲信号。

## Visualization

线性函数像一条直线：

```text
y
│
│      /
│     /
│    /
│___/________ x
```

ReLU 像一条折线：

```text

│
│      /
│     /
│____/________ x
```

Sigmoid 像一条 S 曲线：

```text

│       ______
│     /
│   /
│__/__________ x
```

这些形状不是装饰。它们决定了模型如何处理信号。

## PyTorch Preview

一个最小的非线性网络：

```python
import torch
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(2, 16),
    nn.ReLU(),
    nn.Linear(16, 1),
)

x = torch.tensor([[0.5, -1.0]])
y = model(x)

print(y)
```

如果去掉 `nn.ReLU()`：

```python
model = nn.Sequential(
    nn.Linear(2, 16),
    nn.Linear(16, 1),
)
```

这两层线性层可以合并成一层。

所以 `nn.ReLU()` 不是小细节。它是表达能力的开关。

## Engineering Perspective

在真实工程中，激活函数影响：

- 训练速度；
- 梯度稳定性；
- 数值范围；
- GPU 性能；
- 模型表达能力；
- 推理成本。

Transformer 早期常用 ReLU，后来常用 GELU。许多现代 LLM 使用 SwiGLU 或类似门控结构。

这说明激活函数不是一个已经结束的话题。它仍然是模型架构设计的一部分。

## Failure Modes

### Failure 1: Sigmoid saturation

Sigmoid 在输入很大或很小时会接近 0 或 1。

此时梯度很小，深层网络难以训练。

### Failure 2: Dead ReLU

如果一个 ReLU 神经元长期输出 0，它可能几乎不再更新。

这叫 dead ReLU。

### Failure 3: 非线性太强可能过拟合

表达能力越强，越可能记住训练数据中的噪声。

所以非线性需要正则化、数据、优化和评估配合。

### Failure 4: 金融中的非线性不稳定

市场关系会变。一个非线性模型可能在历史上拟合得很好，却在 regime shift 后失效。

复杂模型不是免费的午餐。

## AI x Finance

金融世界充满非线性。

例如估值：

```text
低估值 + 盈利改善 -> 可能重估
高估值 + 盈利下修 -> 可能剧烈下跌
```

同一个盈利变化，在不同估值区间、利率环境和市场情绪下，效果完全不同。

再如杠杆：

低利率时期，杠杆可能提高 ROE。

利率快速上升时，杠杆可能突然变成风险源。

这都是非线性。

因此，AI 投研系统不能只学线性因子，还要学习条件、阈值和组合效应。

但也要记住：非线性越强，越需要防止过拟合。

## Chapter Summary

本章建立了非线性的核心直觉：

- 线性变换负责重组表示。
- 多个线性变换如果没有非线性，可以合并成一个线性变换。
- 非线性打破这种限制，让模型能够表达曲线、阈值和复杂边界。
- 激活函数是非线性进入神经网络的主要方式。
- ReLU 简单高效，GELU 和 SwiGLU 在现代大模型中非常重要。
- 非线性强大，但也带来过拟合和不稳定风险。

## Master Insight

> 非线性让模型不只是重新排列世界，而是能够弯曲世界。
