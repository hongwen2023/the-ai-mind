## Opening Story: 复杂世界从简单变化开始

想象你手里有一张透明方格纸。

纸上有一个点，表示一家公司。它的位置由两个数字决定：

```text
x = profitability
y = growth
```

现在你想换一种方式看这家公司。你不再只关心盈利能力和增长速度，而是想看：

```text
quality = 0.7 * profitability + 0.3 * growth
risk    = -0.2 * profitability + 0.8 * growth
```

你并没有创造新的原始数据。你只是把原来的坐标重新组合了一次。

这就是线性变换最朴素的样子：

> 用一组简单、稳定、可解释的规则，把一个表示变成另一个表示。

AI 里的很多层也在做类似的事情。它们不断把输入表示变成新的表示。第一层可能提取边缘，下一层组合形状，再下一层组合语义。每一步看起来都很简单，但多步组合之后，模型就能表达复杂结构。

## The One Sentence

> 线性变换的力量不在于一次完成复杂智能，而在于它能稳定、可组合地改变表示。

## Why Before What

为什么要先学习线性？

因为复杂系统通常不能直接理解。我们需要先找到最简单、最稳定、最可计算的变化方式。

线性变换就是这种最基本的变化。

它不会把空间撕裂，也不会突然折叠。它保留直线和平行结构，只做拉伸、压缩、旋转、剪切和组合。

这听起来很有限，但正是这种限制让它变得强大：

- 它容易计算。
- 它容易求导。
- 它容易组合。
- 它能用矩阵统一表示。
- 它是神经网络每一层的骨架。

现代 AI 不是抛弃线性，而是把线性变换与非线性反复组合。

## Feynman Explanation

把线性变换想象成调音台。

调音台有很多输入声道：

```text
鼓
贝斯
吉他
人声
键盘
```

每个输出声道都是输入声道的某种加权组合。

例如：

```text
舞台监听 = 0.8 * 人声 + 0.4 * 吉他 + 0.2 * 键盘
观众音响 = 0.6 * 人声 + 0.7 * 鼓 + 0.7 * 贝斯
```

这就是线性组合。

AI 中的线性层也类似。它接收一组输入特征，然后用权重把它们重新混合，产生一组新的特征。

它不是魔法。它只是非常系统地问：

> 哪些输入应该组合起来，形成新的表示？

## Historical Background

线性思想很早就出现在科学与工程中。

物理学用线性近似理解小范围运动。

统计学用线性回归解释变量关系。

工程学用线性系统分析信号、电路和控制。

机器学习继承了这条路线。早期模型如线性回归、Logistic Regression、SVM，都依赖线性结构。深度学习看似更复杂，但底层仍然大量依赖矩阵乘法和线性变换。

Transformer 里的许多关键步骤也是线性的：

```text
Q = X @ Wq
K = X @ Wk
V = X @ Wv
```

这些不是装饰。它们是把同一个 token 表示投影到不同语义空间：

- Query 空间；
- Key 空间；
- Value 空间。

所以线性变换不是旧时代的工具。它是现代 AI 的基本语言。

## First Principles

一个线性变换必须满足两个核心性质：

```text
T(a + b) = T(a) + T(b)
T(ca) = cT(a)
```

这两句话的意思是：

第一，先相加再变换，等于先变换再相加。

第二，先放大再变换，等于先变换再放大。

这让线性变换非常可预测。

在矩阵形式里，线性变换通常写成：

```text
y = W @ x
```

如果再加上偏置：

```text
y = W @ x + b
```

严格说，加入偏置后叫仿射变换（affine transformation）。但在神经网络语境中，我们通常把 `Linear layer` 理解为：

```text
linear mixing + bias shift
```

也就是先混合，再平移。

## Intuition: 空间如何被线性改变

请想象二维空间里有一个正方形。

线性变换可以把它变成：

- 更宽的矩形；
- 更高的矩形；
- 倾斜的平行四边形；
- 旋转后的形状；
- 被压扁到一条线附近的形状。

但它不能把正方形变成一个圆，也不能把一条直线弯成 S 形。

这就是线性变换的边界：

> 它擅长重组空间，但不能弯曲空间。

这个边界非常关键。它会自然引出下一章的非线性。

## Mathematics: 多个线性变换仍然是线性的

假设有两个线性变换：

```text
h = W1 @ x
y = W2 @ h
```

代入：

```text
y = W2 @ (W1 @ x)
```

矩阵乘法可以结合：

```text
y = (W2 @ W1) @ x
```

令：

```text
W = W2 @ W1
```

于是：

```text
y = W @ x
```

这说明：

> 如果中间没有非线性，两层线性变换可以合并成一层线性变换。

推广到很多层也是一样。

这就是为什么神经网络不能只堆线性层。如果只有线性层，再深也只是一个大的线性变换。

深度学习真正强大的结构是：

```text
Linear
  ↓
Nonlinearity
  ↓
Linear
  ↓
Nonlinearity
  ↓
...
```

线性负责重组表示，非线性负责弯曲空间。

## Visualization

可以把线性变换想象成对整个网格纸的统一操作：

```text
Before:

┌───┬───┬───┐
│   │   │   │
├───┼───┼───┤
│   │ x │   │
├───┼───┼───┤
│   │   │   │
└───┴───┴───┘

After:

╱───╱───╱───╱

│   │ x'│   │
╱───╱───╱───╱
│   │   │   │
╱───╱───╱───╱
```

每个点都一起移动。

这与任意乱改不同。线性变换是一种有组织、有结构的改变。

## Coding Lab: NumPy 中的线性变换

```python
import numpy as np

x = np.array([2.0, 1.0])

W = np.array([
    [2.0, 0.0],
    [0.0, 0.5],
])

y = W @ x

print(y)
```

这个矩阵会把第一维放大 2 倍，把第二维压缩到一半。

再看一个特征混合：

```python
x = np.array([0.8, 0.6])  # profitability, growth

W = np.array([
    [0.7, 0.3],   # quality
    [-0.2, 0.8],  # risk-like dimension
])

y = W @ x

print(y)
```

这段代码的意义不是算两个数字，而是展示：

> 新特征可以由旧特征组合出来。

## PyTorch Preview

```python
import torch
import torch.nn as nn

layer = nn.Linear(in_features=2, out_features=3)

x = torch.tensor([[0.8, 0.6]])
y = layer(x)

print(y)
```

`nn.Linear` 内部做的核心事情就是：

```text
y = x @ W.T + b
```

PyTorch 隐藏了矩阵细节，但没有改变数学本质。

## Engineering Perspective

线性层在工程上有三个巨大优势。

第一，它非常适合 GPU。

现代 GPU 最擅长的就是大规模矩阵乘法。Transformer 之所以能扩展，很大程度上是因为它把大量计算组织成矩阵乘法。

第二，它可微。

线性层的梯度容易计算，适合反向传播。

第三，它通用。

无论输入是图像、文本、音频还是金融特征，只要它们变成向量，就可以通过线性层重组。

所以工程上常见一句话：

> Deep learning is mostly matrix multiplication plus nonlinearities.

这句话不完整，但非常有启发。

## Failure Modes: 线性什么时候不够？

线性变换有边界。

### Failure 1: 不能表达弯曲边界

如果数据分布像月牙形、环形或复杂分段结构，单个线性模型很难拟合。

### Failure 2: 不能自动表达交互

如果结果取决于两个特征的乘积，例如：

```text
risk = leverage * volatility
```

纯线性模型需要额外特征工程才能表达这种关系。

### Failure 3: 多层线性不增加表达力

如果没有非线性，堆很多层只是浪费计算。

### Failure 4: 线性解释可能过度简化现实

在金融中，很多关系不是稳定线性的。估值、流动性、风险偏好、政策变化都可能产生阈值效应和 regime shift。

## AI x Finance

线性模型是金融研究的基础。

多因子模型常常写成：

```text
return = beta1 * value + beta2 * quality + beta3 * momentum + ...
```

这就是线性组合。

它的好处是：

- 可解释；
- 稳定；
- 容易诊断；
- 容易做风险分解。

但它也有局限。

真实市场常常存在非线性：

- 高估值在流动性宽松时可能继续上涨；
- 杠杆在低利率环境中影响较小，在紧缩环境中突然变危险；
- 盈利改善对股价的影响可能取决于预期差。

因此，金融 AI 不能只停留在线性模型，但也不能跳过线性模型。线性是理解复杂模型的基准。

## Chapter Summary

本章建立了线性变换的核心直觉：

- 向量表示对象。
- 矩阵表示变化。
- 线性变换是结构化、可计算、可组合的变化。
- 神经网络中的线性层本质上是特征重组。
- 多个线性变换如果没有非线性，仍然可以合并成一个线性变换。
- 线性模型重要，但不能表达所有复杂现实。

## Master Insight

> 线性变换不是智能的全部，但它是智能系统重组表示的基本动作。
